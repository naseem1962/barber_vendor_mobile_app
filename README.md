# Barber App - Complete Platform

A comprehensive barber shop management platform with AI-powered features for barbers, customers, and administrators.

## Project Structure

```
barber_app/
├── user-web/              # Next.js User Web Application
├── barber-web/            # Angular Barber Web Application
├── user-mobile/           # React Native User Mobile App
├── barber-mobile/         # React Native Barber Mobile App
├── admin-dashboard/       # React.js Admin Dashboard
└── backend/               # Node.js + Express + MongoDB Backend (MVC)
```

## Features

### For Barbers
- AI Hairstyle Recommendation
- Smart Appointment & No-Show Prediction
- AI Pricing Assistant
- Customer History Memory
- AI Business Coach
- Voice Assistant

### For Customers
- AR/AI Face Preview (Try Before You Cut)
- One-Tap Booking with Smart Suggestions
- Hair Growth & Care AI
- Loyalty System with Gamification

### For Admins
- Content Management (Logos, Banners, Privacy Policy, Terms)
- User & Barber Management
- Analytics Dashboard

## Tech Stack

- **Frontend**: Next.js, Angular, React.js, React Native
- **Backend**: Node.js, Express.js, MongoDB
- **Language**: TypeScript (All applications)
- **Architecture**: MVC (Backend)
